# Quick Start Guide

## Solid playgrounds

Experience Solid in your browser by visiting our [interactive playground](https://playground.solidjs.com/).

Additionally, we offer some Stackblitz starters, which provide a convenient web-based development environment. 
You can explore the [JavaScript](https://stackblitz.com/github/solidjs/templates/tree/master/js) version or opt for our [Typescript](https://stackblitz.com/github/solidjs/templates/tree/master/ts) one.

## Creating a Solid application

<Aside>
    **Prerequisites:**
    * *Familiarity with the command line*
    * *Install [Node.js](https://nodejs.org/en)*
</Aside>

Solid offers convenient project templates that can help kickstart your development. 
To get an application running, follow the steps below based on the language you prefer to use.

### For JavaScript projects

1. Create application using a Solid template:

Solid provides a great JavaScript starter template. 
To create your application structure, execute the following command in your terminal:

```bash
npx degit solidjs/templates/js my-app
```

2. Navigate to your application's directory:

```bash
cd my-app
```

3. Install the necessary dependencies:

<TabsCodeBlocks>
<div id="npm">
```bash
npm install
```
</div>

<div id="yarn">
```bash
yarn install
```
</div>

<div id="pnpm">
```bash
pnpm install
```
</div>
</TabsCodeBlocks>

4. Run the application:

```bash
npm run dev
```

This will start the development server. 
Now, open your browser and navigate to `localhost:3000` to see your application running.
